Expandable search bar

An expandable search bar is a CSS-based search box that can expand to either the left or right.1 It requires only HTML and CSS, and the end-user can either tap the search button to expand the search box or hover over it. A working search box that expands when in use requires only HTML and CSS, and the end-user can tap the search button to expand the search box or simply hover over it. A search bar with an icon that expands on click is also available in Foundation 6. To create an expandable search bar in Foundation 6, one can use the "search-field" function to toggle "expand-search" and focus on it if the search field is expanded.
